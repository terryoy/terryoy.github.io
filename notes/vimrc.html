Preferences for Vim

Add below lines to ~/.vimrc:


set smartindent
set tabstop=4
set shiftwidth=4
set expandtab

1. smartindent - Do smart autoindenting when starting a new line for C-like programs. 2. tabstop - Number of spaces that a in the file counts for. 3. shiftwidth - Number of spaces to use for each step of (auto)indent. 4. expandtab - Spaces are used in indents with .

For color scheme preferences: 


colorscheme torte

It is the default black/white scheme.
